### RMS: Relationship Management Skills (Living Edition) / RMS 公開標準版（随時更新）

持続可能で良好な関係を築き、ともに歩む姿勢を土台にマネジメントするスキル体系の、再構成とオープンソース化。

## プロジェクトのビジョン

20年以上の実践と数百回の研修・ワークショップで培われた「関係構築型マネジメントスキル（RMS）」の知見を、MITライセンス（利用・改変・再配布可）で再構築するプロジェクトです。

すべて新規に書き起こし、現代の組織構造やテクノロジーに合わせて「リファクター（再構成）」した「生きた教科書」を目指します。

## このプロジェクトの目玉

本リポジトリでは、特に以下の4点に注力して執筆・公開を進めます。

- RMSの基礎の体系化
  - 20年以上使い続け、効果が実証されたスキルのまとめ。
- 迷いどころ・誤解の解消
  - 多くの受講者が躓いたポイントや、間違った解釈をされやすい部分をていねいに解説。
- 現代的なマネジメントスタイルへのアレンジ（挑戦目標）
  - 多様性や心理的安全性が重視される、現代のチームメンバー・仲間との関わり方に最適化。
  - （実現に時間を要する可能性がありますが、ぜひ補強したいと思っています）
- AIをマネジメントするスキルとしての応用（挑戦目標）
  - 対人スキルを「AI（LLM等）との付き合い方」に応用し、これからの時代に必要な能力を提示。
  - （将来的な展望としての位置づけであり、順次研究・執筆を進めます）
