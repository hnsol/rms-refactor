## インパクト×コスト

インパクト×コストは、効果の大きさと実行負荷を並べて優先順位を考えるテクニックです。

よさそうな案が多いときほど、比較軸が必要です。インパクトとコストを見ることで、どこから手をつけるかを話し合いやすくなります。
