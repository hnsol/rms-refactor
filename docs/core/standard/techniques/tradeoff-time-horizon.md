## トレードオフ（時間軸）

トレードオフ（時間軸）は、短期と中長期で何を優先するかを整理するテクニックです。

今すぐ楽な案が、あとで重くなることがあります。逆に、今は手間でも後から効く案もあります。時間軸をそろえると、合意の前提が見えやすくなります。

## ここで扱うこと

- 短期の影響を確認する
- 中長期の影響を確認する
- 優先する時間軸をそろえる
