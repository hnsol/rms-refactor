## ラポール形成

ラポール形成は、相手が安心して話せる空気を作るためのテクニックです。

相手の速度、言葉の温度、関心に合わせることで、「この人は敵ではない」という感覚を先に作ります。媚びることではなく、対話の入口を整える作法です。

## ここで扱うこと

- 相手の話しやすさを整える
- 警戒を下げる
- 共同作業の入口を作る
