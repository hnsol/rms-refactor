## 懸念のトリアージ

懸念のトリアージは、不安や違和感を種類ごとに切り分けるテクニックです。

懸念をすべて同じように扱うと、議論が止まります。不安なのか、誤解なのか、別件なのかを分けると、今扱うべきことが見えます。
