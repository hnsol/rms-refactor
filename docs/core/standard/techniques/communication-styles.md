## コミュニケーションスタイル

コミュニケーションスタイルは、相手に伝わりやすい順序や情報量を見立てるためのテクニックです。

同じ内容でも、結論を先に聞きたい人もいれば、背景から聞きたい人もいます。相手の受け取り方を見誤ると、話の中身以前でつまずきます。

## ここで扱うこと

- 相手の受け取り方を見立てる
- 説明の順序を調整する
- 伝わらなさを相手のせいにしない
