## 5W2H

5W2Hは、行動を具体化するための確認項目です。

合意したつもりでも、誰が何をいつまでに行うのかが曖昧だと実行は止まります。5W2Hを使うと、行動の輪郭をそろえやすくなります。

## ここで扱うこと

- 実施者と対象を確認する
- 期限と方法を確認する
- 完了条件を確認する
