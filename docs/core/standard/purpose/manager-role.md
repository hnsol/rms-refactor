## マネジャーが役割を果たす

このページでは、RMSにおけるマネジャーの役割を整理します。

マネジャーは、相手を思い通りに動かす人ではありません。チームが成果を出し、チームメンバー・仲間が育ち、合意と実行が前に進む状態をつくる人です。

## ここで扱うこと

- 成果を出すことと人を育てることの両立
- 肩書きではなく役割としてのマネジャー
- 合意と実行を前に進めるための関わり方
