## ビルディング・フェーズ

ビルディングは、真因に対処する必要性や優先順位の認識をそろえるフェーズです。

原因が見えても、その重要さが人によって違って見えていると、行動はそろいません。だから、事実、影響、優先順位を一緒に組み立てます。

## ここで扱うこと

- 事実と影響をつなげる
- 重要性と緊急性の認識をそろえる
- 解決策検討へ進むための土台を作る
