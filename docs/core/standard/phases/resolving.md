## リゾルビング・フェーズ

リゾルビングは、不安や疑問を言語化し、合意や実行を止めているものをほぐすフェーズです。

懸念は邪魔ものではありません。小さな違和感を早めに扱うことで、あとから大きな手戻りになるのを防げます。

## ここで扱うこと

- 不安、誤解、別件を切り分ける
- 今扱う懸念と後で扱う懸念を分ける
- 合意に残った引っかかりを言語化する
