## RMSの大目的

会議が進まない。合意したはずのことが動かない。そんな現場、思い当たりますか？

RMS（Relationship Management Skills）は、そうした停滞をほどき、チームが前に進むための「共通の作法」を整理した体系です。対象は会社だけではありません。家庭や小さなコミュニティでも、人が一緒に動くならそこは組織です。

私たちが目指すのは、**一体感があってごきげんに動いているチーム**です。しんどさはあっても、前進の手応えがあり、関係がこじれにくい状態ですね。では、どうやってそこにたどり着くのでしょうか。

答えはシンプルで、合意と実行が回る状態をつくることです。そのために、問題を定義し、真因を掘り、対策を合意し、行動を続ける。RMSはその一連の流れを、再現できる形に整えます。

この大目的を支える具体的な柱として、RMSには3つのサブ目的があります。概要だけを先に押さえておくと、迷子になりにくくなります。詳しくは [RMSの目的](purpose/index.md) で確認してください。

このあと、RMSの流れを2本立てで案内します。ひとつは、目的の軸で「3つの目的」を整理する話。もうひとつは、実行の軸で「6つのフェーズ」を順にたどる話です。どちらも別々ではなく、行き来しながら理解を深めるための道案内だと思ってください。
